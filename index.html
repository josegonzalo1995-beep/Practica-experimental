TABLA DE REDUCCIÓN DE AMPLITUD POR CICLO

Valores teóricos

Cada fila muestra el pico positivo de un ciclo de oscilación (el pico negativo se considera simétrico, de igual magnitud). Las celdas resaltadas son editables: introduzca en cada una la amplitud medida experimentalmente con la maqueta física. Con dos o más entradas, el sistema infiere ζ por regresión lineal sobre ln(A) vs t y rellena automáticamente las celdas no editadas con la curva de mejor ajuste.

Método del decremento logarítmico. Entre dos picos sucesivos An-1 y An se calcula δ = ln(An-1 / An) y el coeficiente de amortiguamiento real ζ = δ / √(4π² + δ²). Al editar las amplitudes con las mediciones reales tomadas en la maqueta, el ζ inferido se transfiere automáticamente al simulador y la curva teórica se reajusta para reflejar el amortiguamiento experimental.
